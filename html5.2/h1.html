input formaction属性：用于描述表单的action属性 ，属性会覆盖form 的action属性 用于 type=submit 和 type=image

input formenctype:描述表单提交到服务器的数据编码（只对form 的method=post 有效） 覆盖form的enctype属性 用于 type=submit 和 type=image

input formmethod属性，表示提交表单的方式，覆盖form的method属性 用于 type=submit 和 type=image

input formnovalidate:boolean属性 描述input提交表单时无需验证 覆盖form的novalidate属性

input formtarget:属性指定表单提交数据后的网页展示（如：新建，在当前页）覆盖form的target属性

input height和width 用于image类型的input标签的宽高 （如果没设置，浏览器不知道图像大小，则不能预留适当的空间，加载过程中会页面布局改变

input list:规定输入域datalist

input min max:用于数字和日期的input类型的约束

input multiple属性：boolean，允许input 可选择多个之（多张图片）

input pattern属性描述了一个正则表达式用于验证input元素的值。(有text，search，url,tel,email,password等)

input placeholder:提供一种提示(hint),描述所期待的值

input required:boolean 表示必须填写

input step:表示规定的合法数字的间隔
